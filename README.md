# observatoire-sensibilite-feux-foret-68
Observatoire interactif analysant la sensibilité des forêts aux incendies dans le Haut-Rhin (68). Outil de webmapping croisant les données de la sensibilité au feux de forêts (DDT 68) et l'occupation du sol (DIMAP) pour évaluer la vulnérabilité des peuplements forestiers. Développé avec Leaflet.js et Chart.js.
